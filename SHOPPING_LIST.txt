OMEGA-7 SERVO SKULL — SHOPPING LIST
=====================================
Warhammer 40k AI Talking Servo Skull Build
Based on Raspberry Pi 5

CORE COMPUTING
--------------
[X] Raspberry Pi 5 (4GB RAM recommended)
    - Pi 5 is faster and roughly the same price as Pi 4 now
    - 8GB available but unnecessary for this project
[X] MicroSD Card — 32GB minimum, Class 10 / U3 or better
    - 64GB recommended; Piper voice model alone is ~60MB
[X] iUniker 27W PD USB-C Power Supply (B0FHH9K47T)
    - 5.1V @ 5A, GaN, marketed specifically for Raspberry Pi 5
    - Pi 5 REQUIRES 27W; underpowering causes random crashes
[X] Raspberry Pi 5 Active Cooler (official)
    - Fan + heatsink combo designed for Pi 5, very low profile
    - Pi 5 throttles without cooling; position fan away from ReSpeaker mic


AUDIO — INPUT & OUTPUT
-----------------------
[X] UGREEN USB External Sound Card (B01N905VOY)
    - Provides 3.5mm headphone out (→ powered speakers) and 3.5mm mic in (→ nasal mic)
    - USB Audio Class compliant — plug-and-play on Linux, zero driver setup
    - Set AUDIO_OUTPUT_DEVICE and MIC_DEVICE_INDEX in .env to its device index
[X] XMSJSIY Mini Computer Speakers (B0D7L5JQJC)
    - USB powered (can run off Pi USB port), 3.5mm input, volume knob
    - Stereo pair; one inside skull, one behind, or use one mono
[X] 3.5mm electret condenser mic capsule (lavalier / tie-clip type)
    - Mounted in nasal cavity; plugs into UGREEN mic jack
    - Look for a bare capsule or a cheap lavalier with a standard 3.5mm TRS plug


CAMERA — VISION
----------------
[X] Arducam IMX708 Wide Camera Module (B0C5D97DRJ)
    - Sony IMX708, 12MP, 120° FOV, autofocus
    - CSI ribbon cable with 15-to-22 pin adapter included (Pi 5 ready, no extra cable needed)
    - Mounts directly inside skull eye socket; picamera2 driver pre-installed on Pi OS
    - Set CAMERA_ENABLED=true in .env when ready to activate


EYE LEDs — GPIO CONTROLLED
----------------------------
[X] EDGELEC 100pcs 5mm Red Diffused LEDs (B077X95F7C) — use 3 from pack
    - Comes with resistors sized for 6–12V — do NOT use them with Pi GPIO (3.3V)
[X] 220-ohm resistors x3 — for current limiting on GPIO pins 22, 23 & 27 (buy separately)
[X] Jumper wires — male-to-female, 10cm or 20cm
    - For connecting GPIO header to LED circuit
[ ] Mini breadboard or small piece of stripboard/perfboard
    - For mounting the LED + resistor circuit neatly inside skull


PROXIMITY SENSOR
-----------------
[ ] HC-SR04 Ultrasonic Distance Sensor
    - Detects presence/approach up to ~4m; mounts outside skull
    - VCC=5V, GND, Trig (3.3V digital out), Echo (5V — needs voltage divider)
    - Add a 1kΩ + 2kΩ resistor divider on the Echo line to protect Pi GPIO
[ ] 4-pin Dupont ribbon cable (female-to-female, 20–30cm)
    - Routes from sensor outside skull through to GPIO header inside
    - JST-XH 4-pin connector kit is a tidier alternative if you want a locking plug


CANDLE LEDs (TOP OF SKULL)
---------------------------
[X] Self-flickering LED candle bulbs (E10 or bare LED type)
    - Hardware flicker built-in — no Pi control needed
    - Warm white or amber for candle effect
    - Typically sold as "flicker flame LED" bulbs
